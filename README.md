# Home of Boxing Landing Page

Eine statische Landing Page für den Sportclub "The Home of Boxing".

## Lokal ausführen

1. Stelle sicher, dass Python 3 installiert ist.
2. Wechsle in das Projektverzeichnis und starte einen einfachen Webserver:

```bash
python3 -m http.server 8000
```

3. Öffne anschließend im Browser [http://localhost:8000](http://localhost:8000), um die Seite lokal anzusehen.

> Hinweis: Wenn Port 8000 bereits belegt ist, kannst du eine andere freie Portnummer verwenden.
